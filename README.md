Setting up fake server:

    1- $ git clone git@github.com:youssef-from-udacity/fake-api-jwt-json-server.git

    then run commands:

    2- $ npm install
    3- $ npm run start-auth

Setting up Jiblia App project:

    1- $ npm install

    2- $ gulp build

    3- $ gulp watch
